DMARC & SPF Checker - Privacy & Permissions

Permissions Required

activeTab Permission

Required to identify the active tab and its associated domain. This enables the extension to fetch DMARC and SPF records for the website you're currently visiting. Without this permission, the extension cannot analyze or provide insights for the current domain.

dns Permission

Allows the extension to query DNS records directly to retrieve DMARC and SPF details. This permission is essential for providing accurate and real-time DNS analysis of email authentication configurations.

Host Permission (https://dns.google/*)

The extension uses Google's DNS service to fetch DMARC and SPF records securely and reliably. This permission ensures communication with this trusted endpoint for accurate data retrieval.

storage Permission

Used to store the history of analyzed domains and their DMARC/SPF records locally on your device. This enhances user experience by allowing you to revisit past results without re-querying DNS.

Single Purpose Description

This extension serves one specific purpose: to analyze DMARC and SPF policies for any website. By querying DNS records, it provides real-time insights into email authentication configurations, helping users detect misconfigurations or security issues efficiently.

Data Usage & Privacy

Data Collection & Processing

  • All domain analysis is performed through Google's public DNS API
  • DNS query results are stored locally on your device only
  • No personal data is collected or transmitted to third parties
  • History data can be cleared at any time through the extension interface

Remote Code Usage

The extension only makes API calls to Google's DNS server (dns.google) to fetch DNS records. No external or untrusted code execution occurs.
